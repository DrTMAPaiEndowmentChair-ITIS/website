## Project Overview

This project focused on cloud-enabled application development and deployment for drones using the IBM Bluemix Cloud.

### Funding Details
- **Funding Agency**: IBM Pvt. Ltd., Bangalore, India
- **Sanctioned Date**: August 3, 2015
- **Amount**: Rs. 1.50 Lakhs

### Research Team
- **Principal Investigator**: Dr. Manohara Pai M.M. (Dr. T.M.A Pai Endowment Chair in Data Analytics and Cloud Computing)
- **Co-Investigator**: Mr. Sanoop M. (Asst. Professor Senior Scale, Dept. of I & CT, MIT, Manipal)

## Technology Implementation

This innovative project focused on developing:

- **Cloud-based Control Systems** for drone operations
- **Real-time Data Processing** for aerial imagery
- **IoT Integration** for sensor data collection
- **Edge-Cloud Architecture** for efficient processing

### Key Applications

The project delivered several significant drone applications using cloud technology.

## Technical Achievements

This collaborative project with IBM demonstrated several notable innovations:

- **Real-time Processing:** Cloud-based analysis of drone imagery with less than 1s latency
- **Scalable Architecture:** Simultaneous support for multiple drone operations
- **Intelligent Navigation:** Cloud-assisted path planning and obstacle avoidance
- **Data Fusion:** Integration of multiple sensor inputs for comprehensive analysis
- **Low Bandwidth Operation:** Optimized data transmission for remote operations

The cloud-enabled drone applications developed through this research have established new capabilities for aerial data collection and processing, with potential applications in agriculture, urban planning, environmental monitoring, and emergency response scenarios.
