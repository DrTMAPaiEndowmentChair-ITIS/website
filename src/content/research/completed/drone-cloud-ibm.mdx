## Project Overview

This project focused on cloud-enabled application development and deployment for drones using the IBM Bluemix Cloud.

### Funding Details
- **Funding Agency**: IBM Pvt. Ltd., Bangalore, India
- **Sanctioned Date**: August 3, 2015
- **Amount**: Rs. 1.50 Lakhs

### Research Team
- **Principal Investigator**: Dr. Manohara Pai M.M. (Dr. T.M.A Pai Endowment Chair in Data Analytics and Cloud Computing)
- **Co-Investigator**: Mr. Sanoop M. (Asst. Professor Senior Scale, Dept. of I & CT, MIT, Manipal)

## Technology Implementation

This innovative project focused on developing:

- **Cloud-based Control Systems** for drone operations
- **Real-time Data Processing** for aerial imagery
- **IoT Integration** for sensor data collection
- **Edge-Cloud Architecture** for efficient processing
