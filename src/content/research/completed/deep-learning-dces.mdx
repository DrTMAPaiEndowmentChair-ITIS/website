## Project Overview

This project focused on using deep learning and data analytic tools and techniques in Digital Control and Excitation Systems (DCES) applications. It was sponsored by Schneider Electric India Pvt Ltd.

- **Duration**: Till October 2020
- **Amount**: Rs. 11.95 Lakhs
