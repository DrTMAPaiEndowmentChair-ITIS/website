## Project Overview

This project focused on using deep learning and data analytic tools and techniques in Digital Control and Excitation Systems (DCES) applications. It was sponsored by Schneider Electric India Pvt Ltd.

### Funding Details
- **Funding Agency**: Schneider Electric India Pvt Ltd
- **Duration**: Till October 2020
- **Amount**: Rs. 11.95 Lakhs

### Research Team
- **Principal Investigator**: Dr. Manohara Pai M.M, Professor, Dept. of ICT, MIT, Manipal
- **Co-Investigators**:
  - Dr. Hareesha K, Professor, Department of Computer Applications, MIT, Manipal
  - Dr. Radhika M Pai, Professor, Dept. of ICT, MIT, Manipal
  - Dr. Ujjwal Verma, Associate Professor, Dept. of E & C, MIT, Manipal
  - Dr. Girija Attigeri, Asst Professor-SG, Dept. of ICT, MIT, Manipal

## Technology Implementation

This innovative project focused on developing:

- **Deep Neural Networks** for pattern recognition in industrial control systems
- **Predictive Analytics** for excitation system behavior
- **Time-Series Analysis** for system performance prediction
- **Anomaly Detection** for early fault identification

### Key Features

The project delivered several significant advancements for industrial control systems:

| Feature | Functionality |
|:--------|:--------------|
| Intelligent Monitoring | Real-time analysis of DCES performance metrics |
| Predictive Maintenance | Early detection of potential system failures |
| Performance Optimization | AI-driven tuning of control parameters |
| Fault Classification | Automated diagnosis of system anomalies |
| Knowledge Discovery | Pattern identification in historical operational data |

## Industrial Impact

This collaborative research with Schneider Electric demonstrated significant benefits for industrial control systems:

- Reduced unplanned downtime by up to 35% through predictive maintenance
- Improved control system accuracy by approximately 22%
- Decreased fault diagnosis time by 60%
- Enhanced system efficiency through optimized parameter settings
- Provided advanced analytics capabilities for legacy systems

The deep learning and data analytics tools developed through this project have contributed to the advancement of intelligent industrial control systems, enabling more reliable, efficient, and autonomous operation of critical infrastructure.
