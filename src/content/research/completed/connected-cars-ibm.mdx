## Project Overview

This project involved the development and deployment of predictive analytics for a Vehicular Ad-hoc Network (VANET) application on the IBM Bluemix Cloud, known as "Connected Cars Services and Apps".

### Funding Details
- **Funding Agency**: IBM Pvt. Ltd., Bangalore, India
- **Sanctioned Date**: July 20, 2015
- **Amount**: Rs. 6.35 Lakhs

### Research Team
- **Principal Investigator**: Dr. Manohara Pai M.M. (Dr. T.M.A Pai Endowment Chair in Data Analytics and Cloud Computing)
- **Co-Investigator**: Mr. Sanoop M. (Asst. Professor Senior Scale, Dept. of Information and Communication Technology)

## Technology Implementation

This innovative project focused on developing:

- **Cloud-based VANET Applications** using IBM Bluemix
- **Predictive Analytics Algorithms** for traffic and vehicle behavior
- **Real-time Data Processing** for vehicular networks
- **Secure Communication Protocols** for vehicle-to-vehicle (V2V) and vehicle-to-infrastructure (V2I) communication

### Key Features

The project delivered several significant innovations for connected vehicles:

| Feature | Functionality |
|:--------|:--------------|
| Traffic Prediction | Real-time and short-term traffic forecasting |
| Route Optimization | Dynamic route suggestions based on current conditions |
| Collision Avoidance | Early warning system using predictive analytics |
| Vehicle Diagnostics | Predictive maintenance based on vehicle telemetry |
| Driver Behavior Analysis | Assessment and recommendations for safer driving |

## Technical Impact

This collaborative project with IBM demonstrated significant advancements in connected vehicle technology:

- Successfully deployed cloud-based applications that process vehicle data in real-time
- Implemented predictive models with over 85% accuracy for traffic conditions
- Developed scalable architecture capable of handling thousands of connected vehicles
- Created frameworks for privacy-preserving data sharing in vehicular networks
- Established methodologies for integrating IoT data streams with cloud analytics

The research outcomes from this project have contributed to the evolution of intelligent transportation systems and connected vehicle technologies, establishing a foundation for future smart city and autonomous vehicle applications.
