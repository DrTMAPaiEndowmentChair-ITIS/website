## Project Overview

This project focused on Event Driven Programming for Industrial Automation and IoT (Intelligent Technologies), sponsored by Schneider Electric, BGRT, Bengaluru.

- **Sanctioned Date**: January 2024

## Technology Implementation

This project focused on developing:

- **Event-Driven Architecture** for responsive industrial systems
- **IoT Sensor Integration** for real-time monitoring and control
- **Edge Computing Solutions** for distributed processing
- **Intelligent Algorithms** for automated decision-making


The technologies developed through this research contribute to the advancement of Industry 4.0 capabilities and intelligent manufacturing systems.
